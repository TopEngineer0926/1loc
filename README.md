# Single line of code

What's your favorite JavaScript single LOC (line of code)? Let me know!

## Array

* Find the minimum item of an array

~~~ javascript
const min = arr => Math.min(...arr);
~~~

* Get the sum of array of numbers

~~~ javascript
const sum = arr => arr.reduce((a, b) => a + b, 0);
~~~

## Misc

* Swap two variables

~~~ javascript
[a, b] = [b, a];
~~~
